# Подробный 15-дневный план разработки кооперативного 3D-хоррора (спринт)

Вот подробный 15-дневный план разработки (спринт), разбитый на ежедневные микро-задачи. План построен по принципу MVP (Minimum Viable Product): от базовой графики к сетевому коду, а затем к атмосфере и геймплею. В качестве стека используем React + React Three Fiber (R3F) + Rapier (физика) + Socket.io (сеть).

## Этап 1: Локальная 3D-база и физика (Дни 1–4)

**Цель:** Создать основу игры, где можно ходить от первого лица внутри замкнутого пространства.

**День 1: Инициализация проекта и сцены.** Развернуть проект через Vite (React + TypeScript). Установить три базовые библиотеки: three, @types/three, @react-three/fiber, @react-three/drei. Создать базовую сцену `<Canvas>` с кубом-полом и базовым белым светом, проверить рендеринг без ошибок.

**День 2: Контроллер от первого лица (FPS).** Интегрировать компонент `<PointerLockControls />` из библиотеки drei для захвата мыши и вращения камеры. Написать обработчики клавиатуры (WASD) через хук useFrame для расчета вектора движения. Привязать позицию камеры к вектору движения, чтобы реализовать базовое перемещение «в невесомости».

**День 3: Интеграция жесткой физики (Rapier).** Установить @react-three/rapier. Обернуть сцену в компонент `<Physics>`. Заменить куб-пол на физический `<RigidBody type="fixed">`. Создать игрока в виде физической капсулы `<RigidBody type="dynamic" lockRotations>`. Переписать управление WASD так, чтобы менять линейную скорость (setLinvel) этой капсулы, а камеру привязать к её координатам.

**День 4: Дизайн тестовой локации («Коробка»).** Создать в коде или импортировать несколько простых стен-кубов (`<Box>`) с физикой, чтобы сделать лабиринт или закрытую комнату. Проверить коллизии: игрок не должен проходить сквозь стены и должен корректно падать под действием гравитации.

## Этап 2: Сетевой код и мультиплеер (Дни 5–8)

**Цель:** Подключить WebSockets и увидеть других игроков в одной комнате.

**День 5: Настройка WebSocket-сервера.** В корне проекта создать папку /server, инициализировать Node.js проект. Установить express и socket.io. Написать базовый сервер, который слушает порт (например, 3001) и логирует подключение новых сокетов. На клиенте установить socket.io-client и установить соединение с сервером при старте приложения.

**День 6: Менеджмент игроков (Подключение/Отключение).** На сервере создать объект или карту const players = {}, где ключом будет socket.id. При подключении сокета генерировать для него случайные координаты старта и рассылать всем событие player:joined. При отключении — player:left. На клиенте написать логику: хранить список игроков в React-стейт и рендерить напарников в виде простых цветных 3D-капсул/кубов на основе этого стейта.

**День 7: Реальное время (Синхронизация позиций).** На клиенте в хуке useFrame (с частотой ~30 раз в секунду) отправлять на сервер текущие координаты игрока и кватернион (поворот камеры) через сокет: socket.emit('player:move', { pos, rot }). Сервер должен транслировать эти данные всем остальным игрокам в комнате. На клиенте обновлять позиции 3D-моделей напарников. На этом этапе они будут немного "дергаться" — это нормально.

**День 8: Плавность движения (Интерполяция).** Реализовать простейшую линейную интерполяцию (LERP) для напарников. Вместо жесткого присвоения координат использовать метод THREE.MathUtils.lerp. Каждый кадр плавно двигать модель напарника из текущей позиции в целевую сетевую позицию. Движения других игроков должны стать гладкими.

## Этап 3: Атмосфера хоррора и свет (Дни 9–11)

**Цель:** Сделать игру темной, добавить фонарик и страшные звуки.

**День 9: Тьма и тени.** Выключить общий свет (AmbientLight) или сделать его едва заметным (темно-синий, интенсивность 0.02). Включить поддержку теней на Canvas: `<Canvas shadows>`. Задать стенам и полу свойство receiveShadow, а игрокам — castShadow. Добавить в сцену экспоненциальный туман: `<fogExp2 attach="fog" args={['#050505', 0.07]} />`.

**День 10: Реализация фонарика (SpotLight).** Внутрь компонента игрока (прямо к камере) добавить источник света `<spotLight />`. Настроить параметры: узкий угол (angle={Math.PI / 6}), мягкие края (penumbra={0.5}), дальность (distance={20}) и включить тени castShadow. Привязать цель фонарика (spotLight.target) к точке перед камерой, чтобы луч света следовал строго за направлением взгляда игрока.

**День 11: Звуковое окружение.** Добавить базовый жуткий фоновый эмбиент (звук ветра или гул), зацикленный на клиенте. Используя стандартный тег HTML5 `<audio>`, воспроизводить локальный звук шагов, когда игрок зажимает WASD.

## Этап 4: Геймплейные механики и полировка (Дни 12–15)

**Цель:** Добавить сбор лута, интерфейс и протестировать игру с другом.

**День 12: Синхронизация лута (Предметы на карте).** Сервер при старте генерирует массив предметов на карте: const items = [{ id: 1, type: 'scrap', x: 5, z: -3 }]. Клиент при старте скачивает этот массив и рендерит предметы (например, маленькие светящиеся сферы). Написать логику: если игрок подходит к предмету и нажимает E, клиент шлет item:pickup. Сервер удаляет предмет из своего массива и шлет всем команду скрыть этот предмет.

**День 13: Простейший UI (Интерфейс).** Поверх 3D-холста средствами обычного HTML/CSS наложить интерфейс игрока: перекрестие по центру экрана (прицел), индикатор здоровья и счетчик собранного лута. Связать UI с React-состоянием (при подборе предмета счетчик увеличивается).

**День 14: Оптимизация производительности.** Проверить количество полигонов и частоту кадров (FPS). Настроить размеры карт теней (shadow-mapSize), чтобы игра не лагала. Настроить физические хитбоксы — сделать их максимально простыми (капсулы и коробки), чтобы не перегружать процессор.

**День 15: Деплой и финальный тест.** Развернуть бэкенд на бесплатном хостинге (например, Render или Railway), а фронтенд — на Vercel или Netlify. Сбросить ссылку другу, зайти вместе в игру, включить фонарики и проверить синхронизацию ходьбы и сбора предметов.

```
import * as THREE from 'three';
// Импортируем контроллер PointerLock для управления мышью от первого лица
import { PointerLockControls } from 'three/examples/jsm/controls/PointerLockControls.js';
import RAPIER from '@dimforge/rapier3d-compat';

// Исправление №1: передаем пустой объект для инициализации Rapier
RAPIER.init({}).then(() => {
    runGame(RAPIER);
});

function runGame(RAPIER) {
    // 1. Физический мир
    const gravity = { x: 0.0, y: -9.81, z: 0.0 };
    const world = new RAPIER.World(gravity);

    // 2. Сцена и камера
    const scene = new THREE.Scene();
    scene.background = new THREE.Color('#030303'); // Делаем бункер еще темнее
    
    const camera = new THREE.PerspectiveCamera(75, window.innerWidth / window.innerHeight, 0.1, 1000);

    // 3. Рендерер и тени
    const renderer = new THREE.WebGLRenderer({ antialias: true });
    renderer.setSize(window.innerWidth, window.innerHeight);
    renderer.shadowMap.enabled = true;
    // Исправление №2: используем актуальный PCFShadowMap
    renderer.shadowMap.type = THREE.PCFShadowMap; 
    document.body.appendChild(renderer.domElement);

    // 4. Освещение (Направленный свет + слабый эмбиент)
    const ambientLight = new THREE.AmbientLight(0xffffff, 0.02);
    scene.add(ambientLight);

    const dirLight = new THREE.DirectionalLight(0xffffff, 0.6);
    dirLight.position.set(10, 20, 10);
    dirLight.castShadow = true;
    dirLight.shadow.mapSize.width = 1024;
    dirLight.shadow.mapSize.height = 1024;
    scene.add(dirLight);

    // Массив для синхронизации физики с графикой
    const physicsPairs = [];

    // 5. Создаем физический пол
    const floorGeo = new THREE.PlaneGeometry(50, 50);
    const floorMat = new THREE.MeshStandardMaterial({ color: 0x151515, roughness: 0.9 });
    const floorMesh = new THREE.Mesh(floorGeo, floorMat);
    floorMesh.rotation.x = -Math.PI / 2;
    floorMesh.receiveShadow = true;
    scene.add(floorMesh);

    const floorBodyDesc = RAPIER.RigidBodyDesc.fixed().setTranslation(0, 0, 0);
    const floorBody = world.createRigidBody(floorBodyDesc);
    const floorColliderDesc = RAPIER.ColliderDesc.cuboid(25, 0.1, 25);
    world.createCollider(floorColliderDesc, floorBody);

    // 6. Создаем физический куб (Препятствие на карте)
    const cubeGeo = new THREE.BoxGeometry(2, 2, 2);
    const cubeMat = new THREE.MeshStandardMaterial({ color: 0x00ff00, roughness: 0.5 });
    const cubeMesh = new THREE.Mesh(cubeGeo, cubeMat);
    cubeMesh.castShadow = true;
    cubeMesh.receiveShadow = true;
    scene.add(cubeMesh);

    const cubeBodyDesc = RAPIER.RigidBodyDesc.dynamic().setTranslation(0, 6, -5); // Спавним куб чуть впереди
    const cubeBody = world.createRigidBody(cubeBodyDesc);
    const cubeColliderDesc = RAPIER.ColliderDesc.cuboid(1, 1, 1);
    world.createCollider(cubeColliderDesc, cubeBody);
    physicsPairs.push({ mesh: cubeMesh, body: cubeBody });

    // ==========================================
    // 7. СОЗДАЕМ ФИЗИЧЕСКОГО ИГРОКА И КАНАЛЫ УПРАВЛЕНИЯ
    // ==========================================
    
    // Физическое тело игрока (Капсула, чтобы не застревать в углах)
    const playerBodyDesc = RAPIER.RigidBodyDesc.dynamic()
        .setTranslation(0, 2, 0) // Старт над полом
        .lockRotations();       // Запрещаем игроку падать на бок (очень важно!)
    const playerBody = world.createRigidBody(playerBodyDesc);
    const playerColliderDesc = RAPIER.ColliderDesc.capsule(0.5, 0.5); // радиус 0.5, высота 1
    world.createCollider(playerColliderDesc, playerBody);

    // Подключаем управление мышью от первого лица
    const controls = new PointerLockControls(camera, renderer.domElement);

    // Активируем захват мыши при клике по экрану игры
    window.addEventListener('click', () => {
        controls.lock();
    });

    // Обработка клавиатуры WASD
    const keys = { w: false, a: false, s: false, d: false };
    window.addEventListener('keydown', (e) => { if (e.key.toLowerCase() in keys) keys[e.key.toLowerCase()] = true; });
    window.addEventListener('keyup', (e) => { if (e.key.toLowerCase() in keys) keys[e.key.toLowerCase()] = false; });

    // Векторы для расчета направления движения
    const moveDirection = new THREE.Vector3();
    const frontVector = new THREE.Vector3();
    const sideVector = new THREE.Vector3();
    const speed = 6; // Скорость ходьбы игрока

    // 8. Игровой цикл
    function animate() {
        requestAnimationFrame(animate);

        // Шаг физического мира
        world.step();

        // Синхронизируем физические тела (наш зеленый куб) с графикой
        physicsPairs.forEach(pair => {
            const position = pair.body.translation();
            const rotation = pair.body.rotation();
            pair.mesh.position.set(position.x, position.y, position.z);
            pair.mesh.quaternion.set(rotation.x, rotation.y, rotation.z, rotation.w);
        });

        // ЛОГИКА ДВИЖЕНИЯ ИГРОКА (Только если курсор мыши захвачен игрой)
        if (controls.isLocked) {
            // Рассчитываем вектор "вперед/назад" на основе направления взгляда камеры
            frontVector.set(0, 0, Number(keys.s) - Number(keys.w));
            // Рассчитываем вектор "влево/вправо"
            sideVector.set(0, 0, Number(keys.d) - Number(keys.a));

            // Проецируем движения на плоскость пола (чтобы игрок не летал вверх, смотря в небо)
            camera.getWorldDirection(moveDirection);
            moveDirection.y = 0;
            moveDirection.normalize();

            // Создаем финальный вектор скорости
            const targetVelocityX = (moveDirection.x * frontVector.z + camera.up.clone().cross(moveDirection).negate().x * sideVector.z) * speed;
            const targetVelocityZ = (moveDirection.z * frontVector.z + camera.up.clone().cross(moveDirection).negate().z * sideVector.z) * speed;

            // Сохраняем текущую силу гравитации по оси Y, чтобы игрок мог падать
            const currentYVelocity = playerBody.linvel().y;

            // Прикладываем скорость к физическому телу игрока
            playerBody.setLinvel({ x: targetVelocityX, y: currentYVelocity, z: targetVelocityZ }, true);
        } else {
            // Если игра на паузе (курсор отпущен) — игрок останавливается, но продолжает падать под гравитацией
            playerBody.setLinvel({ x: 0, y: playerBody.linvel().y, z: 0 }, true);
        }

        // Привязываем позицию камеры к физическим координатам тела игрока (на уровне глаз)
        const playerPos = playerBody.translation();
        camera.position.set(playerPos.x, playerPos.y + 0.8, playerPos.z);

        renderer.render(scene, camera);
    }

    // Изменение размеров окна
    window.addEventListener('resize', () => {
        camera.aspect = window.innerWidth / window.innerHeight;
        camera.updateProjectionMatrix();
        renderer.setSize(window.innerWidth, window.innerHeight);
    });

    animate();
}
```
